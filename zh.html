研究内容

我的研究目标是构建能够在大规模云集群中部署 LLM 和 DNN 模型的推理系统，并具有最佳性能、效率和可扩展性。

  • Large Language Model System

    • Seving Classic LLM: 由于巨大的内存消耗和不可预测的输出长度，为LLM应用程序提供服务带来了新的挑战。 我们设计了新颖的 LLM 推理系统（qLLM、tgLLM），以最大限度地缩短 LLM 请求的作业完成时间，并最大限度地提高模型吞吐量和资源利用率。 我们还构建了各种推理系统（InferRAG、InferMM）来管理 RAG 和多模态等场景下的计算资源。
    • Serving Specialized LLM: 最近LLM架构的创新也带来了新的挑战。 我们设计了专门的推理系统（SpecInfer、ParaMoE）来优化推测解码和专家混合的推理管道。 此外，我们还研究了一些有趣的主题，例如前向解码、LoRA 服务、kv-cache 优化等。

  • Deep Neural Network System

    • Latency Sensitive Inference: 为了保证良好的用户体验，基于DNN的应用程序通常与延迟目标相关联。 我们设计了各种模型编排系统（Harpagon、DeepLat、TopInfer），通过动态批处理、请求分派和配置解耦等技术，最大限度地降低延迟目标下的服务成本。 我们还构建了各种资源扩展系统（SLOpt、DeepChain），通过 AoT 编译和模型预预热等技术，在突发工作负载下最大化系统吞吐量。
    • Complex Scenario: 鉴于用例，基于 DNN 的应用程序面临着各种部署要求。 我们设计了多级推理系统（Scrooge、Rim、Olympian），通过模型共置和模型提升等技术来管理边缘/云 GPU 集群中的 DNN 模型。 我们还构建了专门的系统（ALPS、HRL）来处理复杂的场景，例如多模式输入和异构硬件。
